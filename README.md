# Mini Nmap Scanner (Python)

This is a beginner-level Python project created on Kali Linux.

## Features
- Scan a specific port
- Check whether the port is OPEN or CLOSED
- Uses Python socket module

## Tools & Technology
- Python 3
- Kali Linux
- Socket Programming

## How to Run
1. python3 mini_nmap.py
2. Enter IP address
3. Enter port number

## Learning Outcome
- Python basics
- Networking concepts
- Port scanning logic
